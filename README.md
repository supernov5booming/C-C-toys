# 编程玩具

&#x20;

## ✨介绍

这是一个用 C/C++ 编写的编程玩具集合，包含多个有趣的算法实现和系统工具项目。每个项目都是独立可运行的，涵盖了从算法可视化到系统编程的多个领域。

<br />

### 🎯 项目特点

- **算法可视化**：使用 raylib 图形库实现算法的实时可视化展示
- **系统编程实践**：包含完整的 Shell 实现和文件压缩工具
- **数据结构与算法**：霍夫曼编码、鸟群算法、空间殖民算法等经典实现
- **跨平台**：基于标准 C/C++ 和 CMake 构建，支持多平台编译

<br />

## 碎碎念

**像搭建积木一般搭建计算机不同领域的项目**

其中一些项目是对于课堂所学知识的复现，力求能以实践反哺，检验知识的获得，另外绝大多数项目是课堂外的拓展以及对其它话题的兴趣，力求拓宽视野，探索新知。我相信兴趣总是走的平稳和长远的那把利剑。

对于有限的人生来说，未知的知识是无限的。

**《庄子・养生主》**

> 吾生也有涯，**而知也无涯**。

要认识到知识是繁重复杂的，在学了越多的东西后越会发现自己的无知，我们首先要尊重知识，同时，各个知识之间是紧密联系的，有的是路径依赖，有的则是知识重叠，另外还有种种的关系，如何把握知识的复杂性与相似性是一个很好的出发点，带着这种思考从中获得启发与感悟。
